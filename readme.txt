Star Plus Addon For Elementor is a versatile Elementor addon that offers many custom widgets. 
== Description ==

The Star Plus Addon For Elementor is a powerful and versatile Elementor addon designed to enhance your website-building experience. This plugin introduces a range of custom widgets that allow you to easily create stunning and interactive designs. Key features include:

Key Features

Button Widgets:

 Customize buttons with various styles, animations, and effects to make your call-to-actions stand out.

Icon Cards: 
Showcase information and content in a visually appealing manner using icon cards, complete with customizable icons, colors, and layouts.

Whether a beginner or an advanced user, Star Plus Addon For Elementor provides the tools you need to build professional and engaging websites effortlessly. Unlock the full potential of Elementor with this feature-rich add-on.


== Installation ==

1. Upload the Star Plus Addon For the Elementor plugin file to the /wp-content/plugins/ directory.

2. Activate the plugin through the 'Plugins' menu in WordPress.

3. Edit page using elementor. Star Plus Addon For Elementor widgets will start working
